## 🚀 Getting Started

### Prerequisites
- No dependencies required!
- Just a modern web browser
- Git (for version control)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/chandanmoshahary/sijoufx-portfolio.git
```

2. Navigate to the directory:
```bash
cd sijoufx-portfolio
```

3. Open `index.html` in your browser or use a local server:
```bash
python -m http.server 8000
# or
npx http-server
```

### Deployment on GitHub Pages

1. Push your code to GitHub
2. Go to repository settings
3. Scroll to "GitHub Pages" section
4. Select `main` branch as source
5. Your site will be live at `https://chandanmoshahary.github.io/sijoufx-portfolio/`

## 🎯 Customization

### Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --bg-primary: #050505;      /* Background color */
    --accent: #00BFFF;          /* Accent color */
    --text-primary: #ffffff;    /* Primary text */
    --text-secondary: #b0b0b0;  /* Secondary text */
}
```

### Statistics
Edit the data-target values in `index.html`:
```html
<div class="stat-number" data-target="50">0</div>
```

### Social Media Links
Update the href attributes in the contact section:
```html
<a href="https://youtube.com/@sijoufx" target="_blank">YouTube</a>
<a href="https://instagram.com/sijoufx" target="_blank">Instagram</a>
```

### Contact Email
Update the email in the form submission function in `script.js`:
```javascript
const mailtoLink = `mailto:your-email@example.com?subject=...`;
```
